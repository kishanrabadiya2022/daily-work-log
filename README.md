## What is React Native?
React Native is a framework used to build **mobile applications (Android & iOS)** using **JavaScript and React**.  
It uses native components like `View` and `Text` instead of HTML tags.

---

## What is Expo?
Expo is a tool that makes React Native development **easy and fast**.  
It helps in:
- Quick project setup
- Running apps without Android Studio
- Testing apps directly on a real mobile device
